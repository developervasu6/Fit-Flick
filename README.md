# 🛍️ Fashion Fit Flick

Fashion Fit Flick is a full-stack e-commerce web application where users
can browse fashion products, search items, add products to the cart,
manage their wishlist, and place orders.

## 🚀 Features

### User Features

-   User Authentication (Login & Signup)
-   Browse Products
-   Search Products
-   Filter Products by Category
-   Product Details Page
-   Add to Cart
-   Wishlist Management
-   Quantity Update in Cart
-   Checkout Process
-   Order Summary
-   Responsive Design

### Admin Features

-   Add Products
-   Update Products
-   Delete Products
-   Manage Product Inventory

## 🛠️ Tech Stack

### Frontend

-   React.js
-   HTML5
-   CSS3
-   JavaScript
-   React Router
-   Axios

### Backend

-   Node.js
-   Express.js

### Database

-   MongoDB
-   Mongoose

### Authentication

-   JWT
-   bcrypt

## ⚙️ Installation

``` bash
git clone https://github.com/developervasu6/Fashion-Fit-Flick.git
cd Fashion-Fit-Flick
```

Install dependencies:

``` bash
cd client
npm install

cd ../server
npm install
```

Create a `.env` file:

``` env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run the project:

``` bash
# Backend
npm run dev

# Frontend
cd ../client
npm start
```
