# 🛍️ Fashion Fit Flick

Fashion Fit Flick is a full-stack e-commerce web application where users can browse fashion products, search items, add products to the cart, manage their wishlist, and place orders. The project is designed with a modern and responsive user interface to provide a smooth online shopping experience.

## 🚀 Features

### User Features
- User Authentication (Login & Signup)
- Browse Products
- Search Products
- Filter Products by Category
- Product Details Page
- Add to Cart
- Wishlist Management
- Quantity Update in Cart
- Checkout Process
- Order Summary
- Responsive Design

### Admin Features
- Add Products
- Update Products
- Delete Products
- Manage Product Inventory

## 🛠️ Tech Stack

### Frontend
- React.js

- CSS3

- React Router
- Axios

### Backend
